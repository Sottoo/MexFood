# MexFood 👋

Este es un proyecto de [Expo](https://expo.dev) creado con [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Empezando

### 1. Instalar dependencias

```bash
npm install
```

### 2. Iniciar el proyecto

Puedes iniciar el servidor de desarrollo con:

```bash
npm start
```

## Comandos Disponibles

| Comando | Descripción |
| :--- | :--- |
| `npm start` | Inicia el servidor de desarrollo de Expo. |
| `npm run android` | Abre la aplicación en un emulador de Android. |
| `npm run ios` | Abre la aplicación en un simulador de iOS. |
| `npm run web` | Abre la aplicación en el navegador web. |
| `npm run lint` | Ejecuta el linter para verificar la calidad del código. |
| `npm run reset-project` | Reinicia el proyecto moviendo el código actual a `app-example`. |

## Estructura del Proyecto

- El desarrollo se realiza principalmente dentro del directorio **app**.
- Este proyecto utiliza [enrutamiento basado en archivos](https://docs.expo.dev/router/introduction).
